Backend highlights:

main.py — FastAPI app with CORS
routes/meals.py — POST /meals/log, GET /meals/summary/{date}, GET /meals, DELETE /meals/{id}
services/claude.py — Claude claude-opus-4-6 analyzes meals, returns structured JSON macros
services/storage.py — Supabase Storage for meal images
To get running:

Supabase: Run supabase/migrations/001_initial.sql in your SQL editor, create a public meal-images bucket in Storage
Backend: cd backend && python -m venv .venv && source .venv/bin/activate && pip install -r requirements.txt && cp .env.example .env (fill keys) → uvicorn app.main:app --reload
Frontend: cd frontend && npm install && cp .env.example .env.local → npm run dev
